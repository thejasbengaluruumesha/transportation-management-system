# Transport Management System

## Overview
A console-based Transport Management System developed in C using structures, linked lists, and dynamic memory allocation.

## Features
- Add Bus
- Add Schedule
- Add Passenger
- Display Bus Details
- Display Schedule
- Display Passengers
- Delete Bus
- Delete Schedule
- Delete Passenger
- Maximum 25 passengers per bus

## Technologies Used
- C Programming
- Structures
- Linked Lists
- Dynamic Memory Allocation

## Data Structures Used
- Structures
- Singly Linked List
- Arrays

## How to Compile

```bash
gcc transport_management.c -o transport
```

## Run

```bash
./transport
```
